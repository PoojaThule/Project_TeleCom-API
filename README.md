1.	Project Title
Telecom Contact List API Automation Project

3.	Objective
The objective of this project is to automate and validate the Telecom Contact List REST APIs using Postman and Rest-Assured. The project ensures that all user and contact management functionalities work correctly with proper authentication and response validation. It verifies API reliability, accuracy, and performance through structured test cases and automated execution.

4.	Tools & Technologies Used
•	API Testing Tool: Postman
•	Automation Framework: Java, Rest-Assured, TestNG, Maven
•	Reporting Tool: Extent Reports
•	Build & Dependency Management: Maven (pom.xml)
•	IDE Used: IntelliJ
•	 API Base URL: https://thinking-tester-contact-list.herokuapp.com
